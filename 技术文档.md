# 技术文档 (Technical Documentation)

## 1. 技术栈

### 后端
*   **框架**: Spring Boot 3.x
*   **ORM**: MyBatis-Plus
*   **数据库**: MySQL 8.0
*   **缓存**: Redis (计划引入)
*   **工具**: Hutool, Lombok, Knife4j (API文档)
*   **Excel处理**: EasyExcel

### 前端
*   **框架**: Vue 3 (Composition API)
*   **构建工具**: Vite
*   **UI 库**: (待定，如 Element Plus / Ant Design Vue)
*   **路由**: Vue Router 4
*   **状态管理**: Pinia / Vuex

## 2. 系统架构
采用经典的分层架构：
*   **表现层 (Controller)**: 处理 HTTP 请求，参数校验。
*   **业务层 (Service)**: 核心业务逻辑，事务控制。
*   **持久层 (Mapper)**: 数据库交互。

## 3. 核心技术实现细节

### 3.1 鉴权机制
目前基于 Token (JWT/Session) 实现。计划升级为 Redis + Token 实现可控的会话管理 (Remember Me)。

### 3.2 数据模型
详见 `lose_and_found.sql` 及 `优化方案.md` 中的变更计划。

### 3.3 文件存储
目前支持阿里云 OSS (见 `AliOssUtil.java`)。
