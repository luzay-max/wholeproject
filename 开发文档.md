# 开发文档 (Development Documentation)

## 1. 项目简介
本项目为校园失物招领系统，旨在方便师生发布和查找失物/招领信息。采用前后端分离架构。

## 2. 快速开始

### 2.1 环境准备
*   JDK 17+
*   Node.js 16+
*   MySQL 8.0
*   Redis (可选，推荐)

### 2.2 后端启动
1.  修改 `src/main/resources/application.yml` 中的数据库配置。
2.  运行 `LostAndFoundApplication.java`。
3.  Swagger 文档地址: `http://localhost:8080/doc.html` (待确认)。

### 2.3 前端启动
1.  进入 `multi-window-vue` 目录。
2.  安装依赖: `npm install`。
3.  启动服务: `npm run dev`。

## 3. 目录结构说明

### 后端 (Spring Boot)
*   `controller`: API 接口层
*   `service`: 业务逻辑层
*   `mapper`: 数据持久层
*   `entity`: 数据库实体
*   `dto`: 数据传输对象 (Request)
*   `vo`: 视图对象 (Response)

### 前端 (Vue 3 + Vite)
*   `src/api`: 接口封装
*   `src/views`: 页面视图
*   `src/components`: 公共组件
