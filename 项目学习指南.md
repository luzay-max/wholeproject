# 项目学习指南

## 项目概览
- **项目名称**：失物招领系统 (Lost and Found System)
- **技术栈**：
  - 后端：Java, Spring Boot 3.1.3, MyBatis-Plus, MySQL, Redis, JWT, 阿里云 OSS
  - 前端：Vue 3.5.22, Pinia, Vue Router, Element Plus, Vite
- **项目类型**：全栈 Web 应用
- **主要功能**：
  - 失物发布与管理
  - 拾物发布与管理
  - 物品认领流程
  - 评论与点赞
  - 荣誉榜系统
  - 管理员后台
  - AI 辅助功能

## 关键知识点

### 1. 后端技术栈与架构
**重要程度**：高
**描述**：基于 Spring Boot 的后端架构，采用分层设计，集成多种中间件和服务。
**详细讲解**：
- **Spring Boot 3.1.3**：提供自动配置、内嵌服务器等特性，简化应用开发
- **MyBatis-Plus**：增强的 MyBatis 框架，提供 CRUD 操作、分页、逻辑删除等功能
- **MySQL**：关系型数据库，存储系统数据
- **Redis**：用于缓存和会话管理
- **JWT**：无状态认证机制，用于用户登录和权限验证
- **阿里云 OSS**：对象存储服务，用于存储图片等文件
- **分层架构**：
  - Controller 层：处理 HTTP 请求和响应
  - Service 层：实现业务逻辑
  - Mapper 层：数据访问层
  - Entity 层：数据模型
  - DTO 层：数据传输对象
  - VO 层：视图对象
**学习建议**：
- 学习 Spring Boot 核心概念和自动配置原理
- 掌握 MyBatis-Plus 的基本用法和高级特性
- 理解 JWT 认证机制的原理和实现
- 学习 Redis 在缓存和会话管理中的应用

### 2. 前端技术栈与架构
**重要程度**：高
**描述**：基于 Vue 3 的前端架构，使用现代前端工具和库。
**详细讲解**：
- **Vue 3**：采用 Composition API，提供更好的代码组织和逻辑复用
- **Pinia**：轻量级状态管理库，替代 Vuex，提供更简洁的 API
- **Vue Router**：官方路由管理器，实现单页应用的页面导航
- **Element Plus**：基于 Vue 3 的 UI 组件库，提供丰富的界面元素
- **Vite**：现代化前端构建工具，提供快速的开发体验
- **Axios**：HTTP 客户端，用于与后端 API 通信
- **项目结构**：
  - api/：API 调用封装
  - components/：可复用组件
  - views/：页面组件
  - router/：路由配置
  - store/：状态管理
  - utils/：工具函数
**学习建议**：
- 掌握 Vue 3 Composition API 的使用
- 学习 Pinia 的状态管理模式
- 理解 Vue Router 的路由配置和导航守卫
- 熟悉 Element Plus 组件的使用
- 了解 Vite 的构建原理和配置

### 3. 系统核心功能模块
**重要程度**：高
**描述**：系统的核心业务功能模块，包括失物管理、拾物管理、认领流程等。
**详细讲解**：
- **失物管理**：发布、编辑、查询丢失物品信息
- **拾物管理**：发布、编辑、查询拾到物品信息
- **认领流程**：用户提交认领申请，上传证明材料，管理员审核
- **评论与点赞**：用户可以对物品信息进行评论和点赞
- **荣誉榜**：根据用户贡献自动生成荣誉榜
- **管理员后台**：管理用户、物品、评论、活动等
- **AI 辅助**：使用 AI 生成物品描述建议
**学习建议**：
- 分析核心业务流程和数据模型
- 理解各功能模块的实现逻辑
- 学习如何设计和实现完整的业务流程

### 4. 数据库设计与优化
**重要程度**：中
**描述**：系统的数据库设计，包括表结构、索引、关系等。
**详细讲解**：
- **表结构**：用户表、失物表、拾物表、认领表、评论表、点赞表、荣誉表等
- **关系设计**：一对一、一对多、多对多关系的设计
- **索引优化**：为常用查询字段创建索引
- **逻辑删除**：使用 MyBatis-Plus 的逻辑删除功能
- **数据迁移**：数据库版本管理和迁移
**学习建议**：
- 学习数据库设计原则和范式
- 掌握 MyBatis-Plus 的代码生成器使用
- 理解索引优化的方法和最佳实践

### 5. 安全与性能优化
**重要程度**：中
**描述**：系统的安全措施和性能优化策略。
**详细讲解**：
- **安全措施**：
  - JWT 认证
  - 密码加密
  - 接口权限控制
  - 输入验证
  - SQL 注入防护
- **性能优化**：
  - Redis 缓存
  - 数据库索引
  - 代码优化
  - 分页查询
  - 异步处理
**学习建议**：
- 学习常见的 Web 安全漏洞和防护措施
- 掌握性能优化的方法和工具
- 了解 Redis 缓存的使用场景和最佳实践

## 项目运行指南

### 后端运行
1. 确保 MySQL 数据库已启动
2. 执行 SQL 脚本创建数据库和表结构
3. 配置 `application.yml` 文件中的数据库连接信息
4. 运行 `LostAndFoundApplication` 类启动应用
5. 访问 `http://localhost:8080/swagger-ui.html` 查看 API 文档

### 前端运行
1. 进入 `multi-window-vue - 2.1` 目录
2. 执行 `npm install` 安装依赖
3. 执行 `npm run dev` 启动开发服务器
4. 访问 `http://localhost:5173` 查看前端应用

## 核心 API 说明

### 认证相关
- `POST /api/auth/login`：用户登录
- `POST /api/auth/register`：用户注册
- `GET /api/auth/info`：获取用户信息

### 失物相关
- `POST /api/lost`：发布失物信息
- `GET /api/lost`：获取失物列表
- `GET /api/lost/{id}`：获取失物详情
- `PUT /api/lost/{id}`：更新失物信息
- `DELETE /api/lost/{id}`：删除失物信息

### 拾物相关
- `POST /api/find`：发布拾物信息
- `GET /api/find`：获取拾物列表
- `GET /api/find/{id}`：获取拾物详情
- `PUT /api/find/{id}`：更新拾物信息
- `DELETE /api/find/{id}`：删除拾物信息

### 认领相关
- `POST /api/claim/apply`：提交认领申请
- `POST /api/claim/confirm`：确认认领
- `GET /api/claim/list`：获取认领列表

### 评论相关
- `POST /api/comments`：发表评论
- `GET /api/comments/{infoId}`：获取评论列表
- `DELETE /api/comments/{id}`：删除评论

### 荣誉榜相关
- `GET /api/honor/board`：获取荣誉榜

1. 全栈 Web 应用的开发流程和架构设计

3. Vue 3 前端开发的现代模式和工具

6. 实际项目的部署和维护
